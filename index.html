HTML

  • Enter your HTML notes here:
  • The head element contains information about the webpage.
  • The body element represent the visible content shown to the user.
  • HTML attributes define additional information about an element.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets (<>).
  • In seemantic HTML elements, the name of the element describe its content and the role it plays on a webpage.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • The git command must precede every Git statement.
  • git checkout: moves the working branch to a new branch
  • git add: adds modifications in the current working branch to the staging area
  • The staging area informs Git that changes will be added in the next commit.
  • git pull: receives a branch's modifications into the local environment
  • The origin keyword indicates the source of the pull will be in the GitHub repo.
  • It is important to always do a git pull to make sure you are on the latest version of the project before working.
